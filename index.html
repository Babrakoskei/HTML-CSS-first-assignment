BOX MODEL

Definition of box model

Box model is used in html tags and css to define web page layout

Box model main properties

  1. width
    Specify width element in content area.
  2. padding
    Separate elements inside the border.
  3. Margin
    Separate elements outside the border.
  4. border
    It goes aroung the padding, specify style of element.
  5. Height
    Specify height of element in content area.
  6. Content area
    The actual element inside the border,padding,margin.

    totalWidth = (width+ Margin + padding + border)
    400+(20*2)+(20*2)+(20*2)+(20*2)+(2*2)+(2*2)
    = 568
